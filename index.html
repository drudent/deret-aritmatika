Website ini berisi kalkulator untuk mencari suku ke-n dan menghitung jumlah suku n pertama dari deret aritmatika.

Bentuk umum deret aritmatika dapat ditulis:
a + (a + d) + (a + 2d) + (a + 3d) + …

    dengan
  • a = suku pertama
  • d = selisih
  • 𝑢𝑛 = (a + (n – 1) d) = mencari suku ke-n
  • S𝑛= n/2 (2a + (n − 1) d) = mencari jumlah suku pertama

Ada yang salah dalam penginputan.

Aritmatika sangat penting dalam kehidupan sehari-hari karena digunakan dalam berbagai situasi, seperti berbelanja, mengatur keuangan, menghitung waktu, dan membuat rencana anggaran. Misalnya, saat menghitung harga barang belanjaan, menghitung waktu perjalanan, atau membagi-bagi makanan di antara teman-teman, kita menggunakan konsep aritmatika.
